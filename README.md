#  Proyecto Tienda 31 Minutos – Módulo 2

Este proyecto corresponde a la actividad final del Módulo 2.  
Consiste en crear una página web tipo e-commerce básica, utilizando HTML, CSS, Bootstrap y JavaScript.

La temática elegida fue **“Los perritos de Mario Hugo”** del programa 31 Minutos.

---

##  ¿Qué incluye este proyecto?

### Página principal (index.html)
- Navbar con navegación.
- Banner principal.
- Sección con tarjetas (cards) de Mario Hugo.
- Lista completa de perritos.
- Sección de productos con imágenes y botones.
- Footer con dos botones animados:
  - Subir al inicio  
  - Volver atrás (con mensaje incluido)

### ✔️ Página del carrito (carrito.html)
- Lista de productos de ejemplo.
- Botón “Finalizar compra” (solo decorativo).
- Botón “Volver atrás”, que lleva siempre a la página principal.
